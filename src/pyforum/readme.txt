Dans ce projet, j'ai développé une application en ligne de commande qui simule un forum de discussion. 
J'ai créé plusieurs classes Python, dont Utilisateur, Forum, Publication et Commentaire, chacune responsable de représenter une partie du forum.
Les utilisateurs peuvent s'inscrire, rejoindre des forums, publier des messages et commenter des publications.
J'ai utilisé des fichiers CSV pour sauvegarder les données simples (utilisateurs, forums) et des fichiers JSON pour les structures plus complexes (publications et commentaires).
J'ai également conçu une interface REPL permettant à l'utilisateur d’interagir avec le système dans le terminal. Pour assurer la persistance des données, j'ai mis en place des méthodes de lecture et d'écriture pour chaque type d'objet. 
Enfin, j'ai structuré mon projet proprement, respecté les bonnes pratiques Git avec un dépôt public sur GitHub, et veillé à versionner régulièrement mes modifications.
